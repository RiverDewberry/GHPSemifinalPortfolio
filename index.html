School Projects
4D Minesweeper
Minesweeper on a 4 by 4 by 4 by 4 hypercube. The hypercube is represented as a grid of grids of cells that can either be mines or empty.
Chess
Two player chess with ascii based graphics. The most notable thing about this project is that the entirety of the board's state (piece types, valid moves, the current player's turn, ect.) is stored in a Uint8Array with a size of 65.
FBLA Project
A project made for the FBLA computer game and simulation programming competitive event. This was a group project where my specific contributions were the art, some of the game logic, and the color correction shader.

Personal Projects
ASCII Art Editor
This is a program I made to create ASCII art. This project uses a system to display ascii text on an html canvas that was later used for ASCII platformer.
ASCII Platformer
This is a platformer style game where three levels are being played simultaneously with one game on each color channel. Due to the fact that the method for displaying the text is the one mentioned above, each character is not actually being drawn separately to each color channel, but rather, this effect is being emulated.
Chromatic (ASCII platformer 2)
This is a sequel to the ASCII Platformer with two main changes: a different display system, and more game mechanics. This project draws text to the canvas pixel by pixel allowing each character to be drawn one color channel at a time.
JSON Parser
This is a parser for json files written completely in C that only needs stdio.h to function properly. This is one of my first decently sized projects in the C programming language.
