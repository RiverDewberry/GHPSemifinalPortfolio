School Projects
Minesweeper on a 4 by 4 by 4 by 4 hypercube. The hypercube is represented as a grid of grids, where every grid is a grid of cells. Every cell can either contain a mine or be empty.

Two-player chess with ASCII-based graphics. The most notable thing about this project is that the entirety of the board's state (piece types, valid moves, the current player's turn, etc.) is stored in a Uint8Array with a length of 65. The controls are arrow keys to move, and space to select peices.

Personal Projects
This is a program I made to create ASCII art. This project uses a system to display ASCII text on an HTML canvas, which I implemented and used for the ASCII platformer.

This is a platformer-style game in which three levels are played simultaneously, with one game on each individual color channel. Because the method for displaying the text uses the ASCII Art Editor mentioned above, each character is not actually drawn to each color channel separately. Instead, the effect is being emulated.

Chromatic (ASCII platformer 2)
This sequel to the ASCII Platformer has two main changes: a different display system and more game mechanics. This project draws text to the canvas pixel by pixel, allowing each character to be drawn in one color channel at a time.
